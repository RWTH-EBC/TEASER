25class SwimmingFacility(NonResidential): 

26 """Archetype swimming facility based on project 

27 'Energieeffizienz in Schwimmbädern - Neubau und Bestand' 

28 

29 Subclass from NonResidential archetype class to represent swimming facilities. 

30 

31 The basic model contains 6 zones 

32 - Swimming_hall 

33 - Changing_rooms 

34 - Shower_and_sanitary_rooms 

35 - Entrance 

36 - Supervisory_room 

37 - Technical_room 

38 And 2 swimming pool: 

39 - Swimmer_pool 

40 - Nonswimmer_pool 

41 

42 Unlike the other building types, in case of the swimming facility the parameter 'net_lead_area' corresponds to the 

43 water area. All dimensions of the swimming facility and its zones are defined according to this input 

44 value for the water area (net_lead_area). 

45 

46 The basis for the building dimensions is mainly 

47 KOK-Richtlinien für den Bäderbau 2013 (KOK guidline for pool construction 2013). 

48 The design of the swimming pool model within the swimming facility is based on 

49 DIN 19643 Treatment of water of swimming pools and baths - Part 1 (2012) 

50 

51 

52 Building parameters 

53 ---------- 

54 

55 parent: Project() 

56 The parent class of this object, the Project the Building belongs to. 

57 Allows for better control of hierarchical structures. If not None it 

58 adds this Building instance to Project.buildings. 

59 (default: None) 

60 name : str 

61 Individual name 

62 year_of_construction : int 

63 Year of first construction 

64 height_of_floors : float [m] 

65 Average height of the buildings' floors 

66 number_of_floors : int 

67 Number of building's floors above ground 

68 net_leased_area : float [m2] 

69 For the swimming pool, this parameter corresponds to the total water area of the whole swimming facility and 

70 is distrubuted to the pools according to normative standards 

71 with_ahu : Boolean 

72 If set to True, an empty instance of BuildingAHU is instantiated and 

73 assigned to attribute central_ahu. This instance holds information for 

74 central Air Handling units. Default is False. 

75 internal_gains_mode: int [1, 2, 3] 

76 mode for the internal gains calculation by persons: 

77 1: Temperature and activity degree dependent calculation. The 

78 calculation is based on SIA 2024 (default) 

79 2: Temperature and activity degree independent calculation, the max. 

80 heatflowrate is prescribed by the parameter 

81 fixed_heat_flow_rate_persons. 

82 3: Temperature and activity degree dependent calculation with 

83 consideration of moisture. The calculation is based on SIA 2024 

84 construction_type : str 

85 Construction type of used wall constructions default is "heavy") 

86 heavy: heavy construction 

87 light: light construction 

88 

89 

90 Note 

91 ---------- 

92 The listed attributes are just the ones that are set by the user 

93 calculated values are not included in this list. Changing these values is 

94 expert mode. 

95 

96 

97 Attributes for zones 

98 ---------- 

99 

100 Zones: 

101 - "Zone 1" # Swimming_hall 

102 - "Zone 2" # Changing_rooms 

103 - "Zone 3" # Shower_and_sanitary_rooms 

104 - "Zone 4" # Entrance 

105 - "Zone 5" # Supervisory_room 

106 - "Zone 6" # Sauna_area 

107 - "Zone 7" # Fitness 

108 - "Zone 8" # Technical_room 

109 

110 zone_area_factors : dict 

111 This dictionary contains the name of the zones as keys and a list for each zone as value, 

112 containing the zone usage from BoundaryConditions json (str), the zone area in m² (float), 

113 the zone volume in m3 (float), the zone length in m (float), the zone width in m (float) 

114 and the zone temperature in Kelvin (float) 

115 

116 outer_wall_names : dict 

117 This dictionary contains a random name for the outer walls, 

118 their orientation and tilt. Default is a building in north-south 

119 orientation) 

120 roof_names : dict 

121 This dictionary contains the name of the roofs, their orientation 

122 and tilt. Default is one flat roof. 

123 ground_floor_names : dict 

124 This dictionary contains the name of the ground floors, their 

125 orientation and tilt. Default is one ground floor. 

126 window_names : dict 

127 This dictionary contains the name of the window, their 

128 orientation and tilt. Default is a building in north-south 

129 orientation) 

130 inner_wall_names : dict 

131 This dictionary contains the name of the inner walls, their 

132 orientation and tilt. Default is one cumulated inner wall. 

133 ceiling_names : dict 

134 This dictionary contains the name of the ceilings, their 

135 orientation and tilt. Default is one cumulated ceiling. 

136 floor_names : dict 

137 This dictionary contains the name of the floors, their 

138 orientation and tilt. Default is one cumulated floor. 

139 

140 

141 Attributes for pools 

142 --------- 

143 basic_pools_dict : dict 

144 This dictionary contains the basic pool types ("Swimmer_pool" and "Nonswimmer_pool") as keys 

145 and a list with basic pool information, containing the type of the pool (str), the area 

146 of the pool in m² (float), the length of the pool in m (float), the width of the pool in m (float) 

147 and the perimeter of the pool in m (float) 

148 

149 """

642 def create_basic_pools(self, water_area): 

643 """ 

644 Create swimming pools for swimming facility archetype. 

645 """ 

646 # pool dict with "Swimmer_pool" and "Nonswimmer_pool" [type, area, length, width] 

647 basic_pool_dic = collections.OrderedDict() 

648 

649 # Pools 

650 main_pool_surface = water_area 

651 beginner_pool_surface = 0 

652 

653 if water_area > 312.5 and water_area <= 582: 

654 beginner_pool_surface = 100 

655 beginner_pool_width = 8 

656 beginner_pool_length = 10 

657 elif water_area > 582: 

658 beginner_pool_surface = 166.7 

659 beginner_pool_width = 10 

660 beginner_pool_length = 16.66 

661 else: 

662 beginner_pool_surface = 0.0 

663 beginner_pool_width = 0.0 

664 beginner_pool_length = 0.0 

665 beginner_pool_perimeter = 2*beginner_pool_width+2*beginner_pool_length 

666 

667 main_pool_surface = water_area - beginner_pool_surface 

668 

669 # Assign respective pool from basic pool areas, which defines the aspect ratio 

670 reference = min([312.5, 415, 830, 1050, 1250], key=lambda \ 

671 x: abs(x - main_pool_surface)) 

672 

673 if reference < 830: 

674 main_pool_length = 25 

675 else: 

676 main_pool_length = 50 

677 main_pool_width = main_pool_surface / main_pool_length 

678 main_pool_perimeter = 2*main_pool_width+ 2*main_pool_length 

679 

680 

681 basic_pool_dic["Swimmer_pool"] = ["Swimmer_pool",main_pool_surface, main_pool_length, main_pool_width, 

682 main_pool_perimeter] 

683 if beginner_pool_surface > 0.0: 

684 basic_pool_dic["Nonswimmer_pool"] = ["Nonswimmer_pool",beginner_pool_surface, beginner_pool_length, 

685 beginner_pool_width, beginner_pool_perimeter] 

686 

687 return basic_pool_dic
